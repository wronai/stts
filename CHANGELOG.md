# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.38] - 2026-02-24

### Changed
- Improved voice model configuration
- Enhanced Polish language support (pl_PL-gosia-medium)

## [0.1.0] - 2025-01-01

### Added
- Initial release of stts (Universal Voice Shell)
- Speech-to-text with Vosk (offline, Polish support)
- Text-to-speech with Piper
- YAML-based model configuration
- Python and Node.js backends
- Microphone input / speaker output pipeline
